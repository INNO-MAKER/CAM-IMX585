Software Installation

Step 1

Offline Driver Compilation

Compile and install the IMX585 kernel driver from source. This ensures compatibility regardless of OS updates.

Package Contents: pkg1-imx585-driver-6.12y-offline.tar.gz

  • imx585-v4l2-driver/ -- Kernel driver source code
  • install.sh -- Automated driver installation script
tar -xzf pkg1-imx585-driver-6.12y-offline.tar.gz cd pkg1-imx585-driver chmod +x install.sh sudo ./install.sh
Step 2

Offline libcamera & rpicam-apps Compilation

Compile libcamera with IMX585 IPA support and rpicam-apps from source. Building from source avoids pre-compiled binary incompatibilities caused by system upgrades.

Package Contents: pkg2-rpicam-libcamera-offline.tar.gz

  • libcamera-imx585/ -- libcamera source with IMX585 IPA support
  • rpicam-apps-imx585/ -- rpicam-apps source
  • build.sh -- Automated build and installation script
tar -xzf pkg2-rpicam-libcamera-offline.tar.gz cd pkg2-rpicam-offline chmod +x build.sh sudo ./build.sh # Full mode with Qt support sudo ./build.sh --lite # Lite mode (minimal deps)

Build time: ~30-40 minutes (full mode) or ~15-20 minutes (lite mode)

Testing the Camera

Verify Detection

List available cameras:

rpicam-hello --list-cameras

Capture Commands

Live Preview:

rpicam-hello -t 0

Capture 4K Image:

rpicam-still -o 4k_image.jpg --width 3856 --height 2180

Record 4K Video (10 seconds):

rpicam-vid -t 10000 --width 3856 --height 2180 -o 4k_video.h264
